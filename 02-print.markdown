
* print 常见的输出

```python
# 打印字符串
print("Hello Python")  # Hello Python

# 逗号自动添加默认的分隔符：空格
print("Hello", "Python") # Hello Python

# 加号表示字符拼接
print("Hello" + "Python")  # HelloPython

# 单词之间用#分隔
print("Hello", "Python", sep="#")  # Hello#Python

# *号 表示重复5遍
print("-" * 5)  # -----

# 默认[print输出会换行，end="" 表示不要换行
print("Hello Python", end="")  # Hello Python

```
